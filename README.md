# CS2 游戏辅助工具

这是一个为 Counter-Strike 2 设计的外部游戏辅助工具，提供了多种功能来增强游戏体验。该工具以直观的图形用户界面呈现，支持多种游戏功能的自定义配置。

## 功能特性

### 核心功能
- **自动扳机 (TriggerBot)**: 自动检测并射击敌人
- **自瞄 (Aimbot)**: 智能辅助瞄准敌人
- **叠加层 (Overlay)**: 显示游戏内信息，包括:
  - 方框显示敌人位置
  - 骨骼追踪
  - 血量条显示
  - 小地图雷达
  - 队友显示
- **兔子跳 (Bunnyhop)**: 自动连续跳跃以提高移动速度
- **无闪光弹 (NoFlash)**: 减少或消除闪光弹效果
- **轮廓发光 (Glow)**: 为敌人(或队友)添加发光效果，使其更容易被发现


### 使用说明
1. 以管理员权限运行程序（推荐）
2. 在设置中启用所需功能并进行自定义配置
3. 启动 Counter-Strike 2
4. 点击"启动"按钮激活功能
5. 使用 Insert 键切换窗口显示/隐藏

## 自动偏移量更新

程序启动时会自动从 [cs2-dumper](https://github.com/a2x/cs2-dumper) 的 GitHub 仓库下载最新的游戏偏移量数据，无需手动更新。

### 工作原理
1. 程序启动时自动从 GitHub 下载最新的偏移量文件
2. 将偏移量文件保存到本地 CS2-external-cheating/Offsets 目录
3. 主程序使用这些偏移量进行游戏内存读取
4. 如果下载失败或文件损坏，程序会记录错误并提示用户
