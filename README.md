# CS2 游戏辅助工具

是一个为 Counter-Strike 2 设计的游戏辅助工具，提供了多种功能来增强游戏体验。

## 功能特性

### 核心功能
- **自动扳机 (TriggerBot)**: 自动检测并射击敌人
- **自瞄 (Aimbot)**: 智能辅助瞄准敌人
- **叠加层 (Overlay)**: 显示游戏内信息，包括:
  - 方框显示敌人位置
  - 骨骼追踪
  - 血量条显示
  - 小地图雷达
- **兔子跳 (Bunnyhop)**: 自动连续跳跃以提高移动速度
- **无闪光弹 (NoFlash)**: 减少或消除闪光弹效果

### 新增功能
- **Vulkan 渲染叠加层**: 使用 Vulkan API 渲染的高性能叠加层，提供更流畅的显示效果
- **自动偏移量生成**: 程序打包后能自动检测并生成所需的内存偏移量


### 渲染引擎
提供了多种渲染选项来显示叠加层：
1. **传统渲染**: 基于 pyMeow 的渲染
2. **OpenGL 渲染**: 使用 OpenGL API 的渲染方式
3. **Vulkan 渲染**: 最新的 Vulkan API 渲染（实验性）

Vulkan 渲染是最新添加的功能，它提供了更好的性能和更低的 CPU 占用率。Vulkan 渲染功能目前处于实验阶段

### 内存管理
- 使用 pymem 库进行高效内存读取
- 实现了智能缓存机制以减少重复内存访问
- 多线程设计确保游戏流畅运行


### 使用说明
1. 以管理员权限运行程序
2. 在设置中启用所需功能
3. 启动 Counter-Strike 2
4. 点击"启动"按钮激活功能

## cs2-dumper 集成

本项目已将 [cs2-dumper](https://github.com/a2x/cs2-dumper) 集成到程序中，无需手动运行。程序启动时会自动调用内嵌的 cs2-dumper.exe 来生成最新的游戏偏移量数据。

cs2-dumper 是一个用于从 Counter-Strike 2 进程中提取偏移量和接口的工具，支持 Windows 和 Linux 系统。

### 工作原理
1. 程序启动时自动检测并运行内嵌的 cs2-dumper.exe
2. cs2-dumper 连接到正在运行的 CS2 进程
3. 提取最新的游戏数据偏移量并保存到 output 目录
4. 主程序使用这些偏移量进行游戏内存读取

### 自动扳机设置
- 自定义触发键
- 可调节射击延迟
- 武器特定配置

### 自瞄设置
- 自定义瞄准键
- 可调节视野和灵敏度
- 武器特定配置

### 叠加层设置
- 可自定义颜色和样式
- 可调节渲染帧率
- 多种显示选项
